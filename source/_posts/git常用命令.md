### 常用命令
```
git status
git add . #这是添加所有文件
git commit -m "注释信息"
git log
git pull origin master
git push origin master #若出现远程仓库与本地仓库不同的情况，需要先pull后push
git remote add origin git@github.com:chengchaoccss/blog.git
ssh-keygen -t rsa -C "1280116143@qq.com"
```
